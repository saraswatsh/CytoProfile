Functions for deeper multivariate insights.

  : Analyze data with Sparse Partial Least Squares Discriminant Analysis

- [`cyt_pca()`](https://cytoprofile.cytokineprofile.org/dev/reference/cyt_pca.md)
  : Analyze Data with Principal Component Analysis (PCA) for Cytokines.
- [`cyt_mint_splsda()`](https://cytoprofile.cytokineprofile.org/dev/reference/cyt_mint_splsda.md)

  Least Squares Discriminant Analysis (sPLS-DA).
